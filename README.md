# sphapp
Tutorial app done in workshop @ ETH SPH 22. April

## requirements

install meteor
```
curl https://install.meteor.com/ | sh
```

## run the server
clone this repo
```
git clone https://github.com/LucNaterop/sphapp
```
run the server on localhost:3000
```
cd bikeable-backend/bikeable-backend
meteor npm install
meteor
```
